## Count Odd Numbers below n


### Description:

Given a number n, return the number of positive odd numbers below n, EASY!

### Example:

```
* 7  -> 3 (because odd numbers below 7 are [1, 3, 5])
* 15 -> 7 (because odd numbers below 15 are [1, 3, 5, 7, 9, 11, 13])
```


### Solution


```Python
def odd_count(n):
    return n // 2
```
